# Kuramoto Oscillators

This repository contains MATLAB code for simulations of Kuramoto-style coupled phase oscillators used in our study of synchronization, wave propagation, and symmetry breaking in active biofilm carpets.

## Reference

If you use this code, please cite:

Altın B, Günay ET, Yaman YI, Ünlü A, Gediz Y, Gedik N, Karadaş B, Başaran M, Kocabaş C, Özdemir Ş, Kocabaş A. *Controlling the synchronization and symmetry breaking of coupled bacterial pili on active biofilm carpets*. eLife. 2026. [doi:10.7554/eLife.107609](https://doi.org/10.7554/eLife.107609.2)

## Contents

* `kuramoto.m` — MATLAB implementation of the Kuramoto oscillator simulation.

## Usage

Open MATLAB in the repository folder and run:

```matlab
kuramoto
```
